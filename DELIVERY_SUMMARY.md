## 🚀 5-MINUTE QUICK START

### Step 1: Export to GitHub (3 min)
```bash
git init
git add .
git commit -m "Beagvs marketplace"
git branch -M main
git remote add origin https://github.com/YOU/beagvs-marketplace.git
git push -u origin main
```

### Step 2: Import to Replit (2 min)
1. Go to replit.com
2. Create → Import from GitHub
3. Paste: `https://github.com/YOU/beagvs-marketplace`
4. Click Import
5. Wait 2-5 minutes
6. Click Run
7. Done! 🎉
